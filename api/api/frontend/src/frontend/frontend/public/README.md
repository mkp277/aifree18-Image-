**For educational/demonstration purposes. Not for commercial or illegal use.**

## Features

- 18+ age verification (demo)
- AI image generation stub *(replace with real API for production)*
- AI chat with a model stub *(replace with real AI model for production)*

## Running locally

**Backend:**
```sh
cd api
npm install
npm start
```

**Frontend:**
```sh
cd frontend
npm install
npm start
```

*Visit [http://localhost:3000](http://localhost:3000) in your browser.*

## Notes

- Update the backend URL in `frontend/src/App.js` or set `REACT_APP_API_URL` to match your deployment.
- Replace stubs with real AI/image generation APIs for production.
